Financial Sentiment Analysis

Description
The analysis aims to perform sentiment analysis on multiple texts, to predict the sentiment of text. The analysis uses the combined (FiQA, Financial PhraseBank) datasets. The model will also be evaluated and improved on. Various methods such as using Pipelines, Latent Dirichlet Allocation and more, will be applied in the analysis.

Installation
Use the package manager pip to install the required packages:
pip install -r requirements.txt

Usage
1.Open the Jupyter notebook in your local machine.
2.Run the cells in order to get the results.
